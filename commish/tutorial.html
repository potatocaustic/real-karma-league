üöÄ Getting Started

What is the Commish Portal?

The Commish Portal is a league management system that gives commissioners the tools they need to run their league effectively. As a commish, you have access to five powerful tools for managing games, players, teams, and more.

Important: Your commish access is league-specific. If you're a commish for the Major League, you'll only have access to Major League tools. The same applies to the Minor League.

Accessing the Portal

  1. Navigate to /commish/dashboard.html in your browser
  2. Log in with your authorized account
  3. You'll see the Commish Dashboard with five tool cards
  4. Use the league switcher (if you have access to multiple leagues) to switch between Major and Minor leagues

üìÖ Manage Games & Scores

This tool allows you to enter lineups, scores, and manage deadlines for weekly matchups.

Setting Deadlines

  1. Click on "Manage Games & Scores" from the dashboard
  2. Expand the "Deadline Tools" section at the top
  3. Select the game date and enter the deadline time (in Central Time)
  4. Click "Set" to save the deadline
Tip: Set deadlines a few hours before the first game of the day to give GMs time to submit their lineups.

Entering Game Scores & Lineups

  1. Select a Season from the dropdown
  2. Select a Week from the dropdown
  3. The games for that week will appear below
  4. Click the "Edit" button on any game card to open the lineup modal
Screenshot: Games list showing game cards with team names, current scores, and Edit buttons

Using the Lineup Modal

  1. Select 6 starters for each team from their roster on the left
  2. The selected players will appear in the "Starters" section
  3. Click on a starter to designate them as Captain (gets 1.5x multiplier)
  4. Enter scores for each player (scores are calculated automatically)
  5. Use the "Adjust scores" button if you need to make manual adjustments
  6. Click "Submit Lineups" to save the lineups
  7. Use "More Actions" dropdown to:
    • Mark game as completed
    • Submit score adjustments
    • Finalize the game (locks it from further edits)
Important: Once you click "Finalize Game," the game is locked and cannot be edited without admin intervention.

Game Status Indicators

  • C (Complete): Game has been finalized
  • R (Ready/Live): Game is in progress or ready for scoring
  • P (Pending): Game hasn't been set up yet

üîÑ Manage Transactions

Log all roster moves including trades, signings, cuts, retirements, and unretirements.

Logging a Trade

  1. Select "Trade" from the Transaction Type dropdown
  2. The Trade Builder section will appear
  3. For each team involved:
    • Select the team from the dropdown
    • Click "Add Player" or "Add Pick" to add assets they're sending
    • For each asset, select its destination team
  4. Click "+ Add Another Team" if the trade involves more than 2 teams
  5. Add optional notes about the trade
  6. Click "Log Transaction" to submit
Screenshot: Trade Builder showing two teams with players and picks being exchanged
Trade Tips:
  • Make sure every asset has a destination team selected
  • For 3+ team trades, carefully verify each asset's destination
  • Use the notes field to record trade details or conditions

Free Agent Signing

  1. Select "Free Agent Signing" from the dropdown
  2. Choose the signing team
  3. Type the player's handle in the search box (it will auto-complete)
  4. Add optional notes
  5. Click "Log Transaction"

Cutting a Player

  1. Select "Cut Player" from the dropdown
  2. Optionally filter by team to narrow down the player list
  3. Select the player to cut from the dropdown
  4. Add optional notes
  5. Click "Log Transaction"

Retirement & Unretirement

Retirement:

  1. Select "Retirement"
  2. Optionally filter by team
  3. Select the retiring player
  4. Click "Log Transaction"

Unretirement:

  1. Select "Unretirement"
  2. Select the player coming out of retirement
  3. Select which team they're joining
  4. Click "Log Transaction"
Success! After logging a transaction, you'll see a green success message. You can immediately log another transaction or click "Reset Form" to clear the form.

üë• Manage Players

Edit player information, team assignments, and manage player statuses.

Finding a Player

  1. Select a Season from the dropdown
  2. Use the search box to filter players by handle
  3. The player list will update in real-time as you type
  4. Click "Edit" on any player card to open the edit modal
Screenshot: Player cards showing player handles, team affiliations, and Edit buttons

Editing Player Information

In the player edit modal, you can modify:

  • Player Handle: Their display name
  • Current Team: Which team they're on
  • Player Status:
    • ACTIVE - Currently playing
    • RETIRED - No longer playing
    • INACTIVE - Temporarily not playing
    • FREE_AGENT - Not on any team
  • Accolades:
    • Rookie - First season player
    • All-Star - All-star selection
Note: Player ID is read-only and cannot be changed through the commish portal.

Creating a New Player

  1. Click the "Create New Player" button at the top right
  2. The modal will open in "Create" mode
  3. Enter the player's handle
  4. Select their team
  5. Set their status (usually ACTIVE for new players)
  6. Check "Rookie" if they're in their first season
  7. Click "Save Player Changes"
Remember: When a player is cut or signs with a new team, update their team assignment here to keep rosters accurate.

üèÜ Manage Teams

Update team information including names, conferences, and GM assignments.

Editing Team Information

  1. Select a Season from the dropdown
  2. Find the team you want to edit
  3. Click "Edit" on the team card
  4. In the modal, you can update:
    • Team Name
    • Conference (varies by league)
    • GM Handle (their display name)
    • Color Override (hex color for the team)
  5. Click "Save Team Changes"
Screenshot: Team edit modal showing editable fields
About Conferences:
  • Major League: Western and Eastern conferences
  • Minor League: Uses different conference structure

Team Rebranding

If a team wants to change their name and ID (full rebrand):

  1. Click "Rebrand" on the team card
  2. Enter the new team name
  3. Enter the new team ID (2-4 uppercase letters/numbers, e.g., "LAL", "BOS")
  4. Double-check all information
  5. Click "Submit Rebrand"
Critical: Rebranding is permanent and updates all historical records, players, and draft picks associated with that team. This action cannot be easily undone!

Color Overrides

By default, team colors are detected from their logo. To manually set a color:

  1. Click the color picker in the edit modal
  2. Select your desired color
  3. Click "Save Team Changes"

üìÑ Generate Reports

Generate formatted reports for lineup announcements, deadlines, and Game of the Day (GOTD) voting.

Available Reports

1. Lineups Deadline Report

Generate a deadline reminder with the day's matchups.

  1. Click to expand the "Lineups Deadline Report" section
  2. Select the Game Date
  3. Enter the Deadline Time (e.g., "12:05p ET")
  4. Click "Generate Report"
  5. The formatted report appears below
  6. Click "Copy to Clipboard" to copy the text
  7. Paste into Discord or your announcement channel
Screenshot: Generated deadline report showing matchups and deadline time

2. Vote GOTD Report

Create a Game of the Day voting announcement.

  1. Expand the "Vote GOTD Report" section
  2. Select the Game Date
  3. Click "Generate Report"
  4. A formatted list of the day's matchups appears
  5. Copy and share for voting

3. Daily Lineups Report

Publish the official lineups for the day's games.

  1. Expand the "Daily Lineups Report" section
  2. If there are multiple games, you'll see a GOTD selector
  3. Select which game is the Game of the Day
  4. Click "Generate Report"
  5. The formatted lineups appear, with GOTD highlighted
  6. Copy and share the lineups
Pro Tip: Generate these reports in order:
  1. Lineups Deadline Report (before deadline)
  2. Vote GOTD Report (after deadline, before lineups)
  3. Daily Lineups Report (after GOTD voting closes)
Time Saver: The deadline time auto-populates based on the earliest game time for the selected date!

Common Questions

Q: I can't see Minor League games. Why?

A: Make sure you're using the league switcher to view the Minor League. Your access must be granted for that specific league.

Q: Can I undo a finalized game?

A: No, finalized games are locked. Contact an admin if you need to make changes to a finalized game.

Q: What happens when I log a trade?

A: The transaction is recorded, but you'll still need to manually update player team assignments in "Manage Players."

Q: How do I change a GM's login credentials?

A: GM User IDs and authentication are admin-only features. Contact an admin for these changes.
